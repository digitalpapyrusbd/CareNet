# Caregiver Platform

A digital solution connecting families with verified caregivers in Bangladesh.

## Tech Stack
- Frontend: Next.js 15
- Backend: NestJS
- Database: PostgreSQL (Prisma)

## Setup
```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm run start:dev
```
